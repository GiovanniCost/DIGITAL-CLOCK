<h1>Relógio Mundial</h1>
<br>
<h2>Uma aplicação web simples e elegante que exibe as horas em tempo real de diferentes fusos horários ao redor do mundo. Ideal para quem precisa acompanhar horários internacionais de forma rápida.</h2>
<br>
<h2>Tecnologias Utilizadas:</h2>
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JAVASCRIPT</li>
</ul>
<br>
<h2>Funcionalidades:</h2>
<ul>
  <li>Exibição da hora local atualizada segundo a segundo.</li>
  <li>Seleção de diferentes cidades/fusos horários (ex: Londres, Nova York, Tóquio).</li>
  <li></li>
</ul>
